# Idea Funnel (Now / Next / Later / Parking Lot)

Amaç: Yeni fikirler geldiğinde odağı kaybetmeden yönetmek.

## Kullanım Kuralı
- Her yeni fikir önce bu funnel'a düşer.
- Doğrudan geliştirmeye yalnızca **Now** kolonundakiler alınır.
- Aynı anda en fazla 2 "Now" maddesi açık olur.

## Now (aktif odak)
- Prompt & task quality hardening (tamamlandı)
- Task sync/traceability stabilite testleri (tamamlandı)

## Next (yakın dönem)
- Security scan turu (prompt bazlı)
- AI Governance v2 (operation-level model routing)

## Later (değerli ama acil değil)
- Incremental analysis precision v2 (AST/structured impact)
- Task quality metrics dashboard trend kartları
- ScopeSmith Native Work Mode (Jira/GitHub olmadan opsiyonel SoR modu)

## Parking Lot (tetiklenince ele al)
- Managed Agent UX expansion (ADR-010 guardrail sonrası)
- Branch protection automation platform-specific iyileştirmeler
- Mikro refactor fikirleri
